# Signal Registry Source Application Canonic Codes

## Description
This repo introduces signal source canonic and cross-platform application codes for various programming languages.

## Requirements
Each language and framework has its own dependenices. Requirements are checked with **'boostrap.sh'** script.

## Usage
Just enter language/framework directory and run following commands in order.
```bash
# check and install dependecies
bash bootstrap.sh dependencies

# build source application
bash bootstrap.sh build

# run source application
bash bootstrap.sh run
```
